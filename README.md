# Unybay

Projeto de E-commerce da marca Unybay. Construído com React, TypeScript, Vite e Tailwind CSS.

## Instalação e Execução

Para rodar este projeto na sua máquina:

1. Certifique-se de estar na pasta do projeto:
   ```bash
   cd unybay
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Inicie o servidor local:
   ```bash
   npm run dev
   ```

4. Acesse no seu navegador o endereço que aparecerá no terminal (geralmente `http://localhost:5173/`).

## Tecnologias Utilizadas
- **React**: Biblioteca principal para interfaces.
- **Vite**: Ferramenta de compilação e servidor local rápido.
- **Tailwind CSS**: Estilização baseada em classes utilitárias.
- **TypeScript**: Maior segurança e inteligência no código JavaScript.
